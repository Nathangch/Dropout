# 🐐 Dropout: A Missão de Japeth

**Dropout** é um jogo de corrida infinita (endless runner) 2D desenvolvido em Python com a biblioteca **Pygame**. Inspirado na fluidez e estética de clássicos como *Alto's Adventure*, o jogo coloca você no papel de um personagem icônico em uma jornada mística e musical.

---

## 📖 A História

Você joga como **Japeth**, o famoso bode das montanhas do filme *"Deu a Louca na Chapeuzinho"* (Hoodwinked!). 

Japeth foi amaldiçoado por uma bruxa do pântano e agora é condenado a **cantar tudo o que diz**. Para quebrar essa maldição e recuperar o silêncio (e a sanidade), ele deve atravessar três biomas perigosos em busca do lendário **Berrante Silencioso**. 

A jornada não será fácil: lobos, escorpiões e até golens de gelo guardam o caminho, e uma avalanche massiva persegue Japeth nas montanhas nevadas!

---

## 🎮 Funcionalidades

- **Três Biomas Únicos**:
    - **Planícies**: Onde tudo começa, com terrenos fluidos e lobos selvagens.
    - **Deserto**: Clima seco, areia movediça e escorpiões perigosos.
    - **Neve**: Terreno montanhoso e escorregadio, onde golens de gelo e uma avalanche implacável aguardam o herói.
- **Sistema de Avalanche**: Sinta a pressão de uma parede de neve massiva que persegue você durante a fase final do bioma de neve!
- **Mecânicas de Movimento**: Salte sobre buracos e use o **Dash** para ganhar velocidade e escapar da morte.
- **Narrativa Integrada**: Telas de introdução e encerramento que contam a saga de Japeth.
- **Gráficos Procedurais**: Chão gerado dinamicamente com ondas senoidais para uma experiência sempre diferente.

## 🚀 Como Executar

### Pré-requisitos
Certifique-se de ter o Python instalado. Se não tiver o Pygame, instale-o via terminal:

```bash
pip install pygame
```

### Rodando o Jogo
Navegue até a pasta do projeto e execute:

```bash
python main.py
```
